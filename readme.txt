MostForce Live Video Chat for Wordpress plugin that allow visitors establish live video chat in browser without download and plugin installation.




== Description ==

MostForce Live Video Chat is an all new way to let your website visitors contact you in real-time, it's built base on the HTML5, establish the live video chat in browser just one click without download and installation, help online visitors find what they need before they leave your site.

Try MostForce Live Video Chat free for 15 days! Sign up here: https://www.mostforce.com/signup.html

Take a tour to see why you need MostForce Live Video Chat on your website: http://www.mostforce.com/en/tour.html




== Installation ==

1. Upload `mostforce-live-video-chat` directory to Wordpress plugins directory (`/wp-content/plugins/`)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click the 'MostForce Live Video Chat' menu on the left.
4. Follow the instructions.
